# Supermarket Billing System

A simple Console-based Supermarket Billing System written in C.

## How to Run

### Option 1: Double-Click
1. Double-click **`build.bat`** to compile the project.
2. Once finished, double-click **`supermarket.exe`** to run the app.

### Option 2: Command Line
Open a terminal in this folder and run:
```cmd
.\build.bat
.\supermarket.exe
```

## Features
- **Add Products**: Manage inventory in `products.txt`.
- **Cart System**: Add/Remove items with stock validation.
- **Billing**: Automatic discount calculation and receipt generation.
- **Logs**: Sales are logged in `logs.txt`.
